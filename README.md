# Room Occupancy Detector

## Dependencies

- Python 3.6+
- OpenCV
- Pandas
- Ultralytics
- Paho-MQTT
- Dotenv

## Usage

```terminal
$ python3 main.py -h
usage: main.py [-h] [-v VIDEO]

options:
  -h, --help              show this help message and exit
  -v VIDEO, --video VIDEO video source (none: webcam)
```

> **Note:** Run from inside `src/`
